# examen-java-frontend

Frontend React para login contra el backend Java.

## Requisitos

- Node.js 18+
- Backend corriendo en `http://localhost:8080`

## Ejecutar frontend

```bash
npm install
npm run dev
```

Abre `http://localhost:5173`.
